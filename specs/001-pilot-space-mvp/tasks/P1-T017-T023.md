## T022: Create quality gate script

**Complexity**: 🟢 3/20 | **Priority**: P1 | **Story**: INFRA
**Blocked by**: T006, T007, T014, T015

### Objective

Create unified quality gate script for both backend and frontend.

### Acceptance Criteria

- [ ] AC1: `scripts/quality-check.sh` exists
- [ ] AC2: Backend checks: `ruff check`, `pyright`, `pytest --cov`
- [ ] AC3: Frontend checks: `pnpm lint`, `pnpm type-check`, `pnpm test`
- [ ] AC4: Exit code reflects pass/fail status
- [ ] AC5: Colored output for results
- [ ] AC6: Summary report at end
- [ ] AC7: Can run individual components with flags: `--backend`, `--frontend`
- [ ] AC8: Script is executable: `chmod +x scripts/quality-check.sh`

### Target Files

- `scripts/quality-check.sh`

### Script Structure

```bash
#!/bin/bash
set -e

# Colors
RED='\033[0;31m'
GREEN='\033[0;32m'
NC='\033[0m'

# Flags
RUN_BACKEND=false
RUN_FRONTEND=false

# Parse arguments
while [[ $# -gt 0 ]]; do
    case $1 in
        --backend) RUN_BACKEND=true; shift ;;
        --frontend) RUN_FRONTEND=true; shift ;;
        *) shift ;;
    esac
done

# Default: run both
if [[ "$RUN_BACKEND" == false && "$RUN_FRONTEND" == false ]]; then
    RUN_BACKEND=true
    RUN_FRONTEND=true
fi

FAILED=0

# Backend checks
if $RUN_BACKEND; then
    echo "🐍 Running backend quality checks..."
    cd backend
    uv run ruff check . || FAILED=1
    uv run pyright || FAILED=1
    uv run pytest --cov=. --cov-fail-under=80 || FAILED=1
    cd ..
fi

# Frontend checks
if $RUN_FRONTEND; then
    echo "⚛️ Running frontend quality checks..."
    cd frontend
    pnpm lint || FAILED=1
    pnpm type-check || FAILED=1
    pnpm test --run || FAILED=1
    cd ..
fi

# Summary
if [[ $FAILED -eq 0 ]]; then
    echo -e "${GREEN}✅ All quality checks passed${NC}"
else
    echo -e "${RED}❌ Quality checks failed${NC}"
    exit 1
fi
```

## T023 [P]: Configure GitHub Actions CI

**Complexity**: 🟢 4/20 | **Priority**: P1 | **Story**: INFRA
**Parallel with**: T022

### Objective

Configure GitHub Actions for continuous integration.

### Acceptance Criteria

- [ ] AC1: `.github/workflows/ci.yml` exists
- [ ] AC2: Triggers: push to main, pull requests
- [ ] AC3: Backend job: Python 3.12, uv, ruff, pyright, pytest
- [ ] AC4: Frontend job: Node 20, pnpm, lint, type-check, test
- [ ] AC5: PostgreSQL service for backend tests
- [ ] AC6: Coverage upload to Codecov (optional)
- [ ] AC7: Cache for uv and pnpm dependencies
- [ ] AC8: Status checks required for merge

### Target Files

- `.github/workflows/ci.yml`

### Workflow Structure

```yaml
name: CI

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  backend:
    runs-on: ubuntu-latest
    services:
      postgres:
        image: supabase/postgres:16.1.0
        env:
          POSTGRES_PASSWORD: postgres
          POSTGRES_DB: pilot_space_test
        ports:
          - 5432:5432
        options: >-
          --health-cmd pg_isready
          --health-interval 10s
          --health-timeout 5s
          --health-retries 5

    steps:
      - uses: actions/checkout@v4
      - uses: astral-sh/setup-uv@v2
      - name: Set up Python
        run: uv python install 3.12
      - name: Install dependencies
        run: cd backend && uv sync
      - name: Lint
        run: cd backend && uv run ruff check .
      - name: Type check
        run: cd backend && uv run pyright
      - name: Test
        run: cd backend && uv run pytest --cov=. --cov-report=xml
        env:
          DATABASE_URL: postgresql://postgres:postgres@localhost:5432/pilot_space_test

  frontend:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: pnpm/action-setup@v2
        with:
          version: 9
      - uses: actions/setup-node@v4
        with:
          node-version: 20
          cache: 'pnpm'
          cache-dependency-path: frontend/pnpm-lock.yaml
      - name: Install dependencies
        run: cd frontend && pnpm install
      - name: Lint
        run: cd frontend && pnpm lint
      - name: Type check
        run: cd frontend && pnpm type-check
      - name: Test
        run: cd frontend && pnpm test --run
```
