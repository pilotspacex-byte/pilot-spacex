**Complexity**: 🟢 4/20 | **Priority**: P1 | **Story**: INFRA

### Objective

### Acceptance Criteria

- [ ] AC2: Health check endpoint at `/health` returns `{"status": "healthy"}`
- [ ] AC3: API docs available at `/docs` (Swagger) and `/redoc`
- [ ] AC4: CORS middleware configured (placeholder for production origins)
- [ ] AC5: API router mounted at `/api/v1`
- [ ] AC6: App can start with `uvicorn pilot_space.main:app --reload`

### Target Files

- `backend/src/pilot_space/main.py`

### Code Structure

```python
"""Pilot Space API - AI-Augmented SDLC Platform."""
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware

app = FastAPI(
    title="Pilot Space API",
    description="AI-Augmented SDLC Platform with Note-First Workflow",
    version="0.1.0",
)

# CORS middleware
app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:3000"],  # Frontend dev server
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

@app.get("/health")
async def health_check() -> dict[str, str]:
    """Health check endpoint."""
    return {"status": "healthy"}

# Router mounting will be added in T045
```
