### Velocity & Burndown Computation

```python
# Velocity
velocity = completed_points / weeks_in_cycle
historical_velocity = avg(last_5_cycles.velocity)

# Burndown (daily)
ideal_remaining = total_points * (days_remaining / total_days)
actual_remaining = sum(incomplete_issues.points)
```
