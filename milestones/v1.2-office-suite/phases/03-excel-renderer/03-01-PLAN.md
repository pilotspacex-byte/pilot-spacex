       ```
       cd frontend && pnpm install --save https://cdn.sheetjs.com/xlsx-0.20.3/xlsx-0.20.3.tgz
       ```

    2. Create `XlsxRenderer.tsx` following CsvRenderer pattern closely. Component signature:
       ```typescript
       interface XlsxRendererProps {
         content: ArrayBuffer;
       }
       export function XlsxRenderer({ content }: XlsxRendererProps)
       ```

    3. Parsing strategy — use setTimeout deferred parse (NOT Web Worker) per user decision:
       - Use `React.useState` for: `parsedWorkbook` (WorkBook | null), `isParsing` (boolean), `activeSheet` (string), `error` (boolean)
       - In a `React.useEffect` on `content`, set `isParsing = true`, then `setTimeout(() => { ... }, 0)` wrapping:
         ```typescript
         import * as XLSX from 'xlsx';
         const wb = XLSX.read(content, { dense: true });
         ```
       - After parse: set workbook, set activeSheet to `wb.SheetNames[0]`, set isParsing = false
       - On error: set error = true, isParsing = false

    4. Sheet data extraction — derive from activeSheet using `React.useMemo`:
       ```typescript
       const sheetData = React.useMemo(() => {
         if (!parsedWorkbook || !activeSheet) return null;
         const ws = parsedWorkbook.Sheets[activeSheet];
         const jsonRows: unknown[][] = XLSX.utils.sheet_to_json(ws, { header: 1, raw: false, defval: '' });
         const [headerRow, ...dataRows] = jsonRows;
         const totalRows = dataRows.length;
         const truncated = totalRows > MAX_ROWS;
         return {
           headers: (headerRow ?? []) as string[],
           rows: dataRows.slice(0, MAX_ROWS) as string[][],
           totalRows,
           truncated,
         };
       }, [parsedWorkbook, activeSheet]);
       ```
       Use `header: 1` (array-of-arrays), `raw: false` (formatted strings), `defval: ''` (empty cells as empty string).
       Apply `MAX_ROWS = 500` constant.

    5. Render structure (follow CsvRenderer layout):
       - If `isParsing`: show spinner (same markup as FilePreviewModal loading state)
       - If `error`: show `<DownloadFallback>` with reason="error"
       - Otherwise render `<div className="flex flex-col h-full">`:
         - Truncation banner (if truncated): same text as CsvRenderer — "Showing 500 of N rows. Download for full data."
         - `<ScrollArea className="flex-1">` wrapping `<Table>`:
           - `<TableHeader>` with `<TableRow>` mapping headers to `<TableHead>`
           - `<TableBody>` mapping rows to `<TableRow>` with alternating `bg-muted/30` (same as CsvRenderer)
         - Sheet tab bar at bottom (below ScrollArea, above any truncation info):
           ```tsx
           {parsedWorkbook && parsedWorkbook.SheetNames.length > 1 && (
             <div className="flex items-center gap-1 px-3 py-2 border-t overflow-x-auto shrink-0">
               {parsedWorkbook.SheetNames.map((name) => (
                 <button
                   key={name}
                   onClick={() => setActiveSheet(name)}
                   className={cn(
                     'px-3 py-1 text-xs rounded-md whitespace-nowrap transition-colors',
                     activeSheet === name
                       ? 'bg-primary text-primary-foreground'
                       : 'bg-muted text-muted-foreground hover:bg-muted/80'
                   )}
                 >
                   {name}
                 </button>
               ))}
             </div>
           )}
           ```
         - Layout order top-to-bottom: truncation banner | ScrollArea with table | sheet tab bar

    6. Import `cn` from `@/lib/utils`. Import `DownloadFallback` from `./DownloadFallback`. Import shadcn Table components and ScrollArea same as CsvRenderer.
