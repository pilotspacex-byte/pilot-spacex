Proposed Solution

For the authentication refactor:

  • Implement OAuth2 with PKCE for all social providers
  • Add magic link option for enterprise users
  • Show clear error states with retry options
  • Extend session to 7 days with refresh tokens

Technical Approach

We'll use the existing auth service but add a new OAuth flow handler. The token refresh mechanism needs updating to support longer sessions without compromising security.
