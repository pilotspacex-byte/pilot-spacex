Proposed Solution

For the authentication refactor, I'm thinking we should:

  • Implement OAuth2 with PKCE for all social providers
  • Add magic link option for enterprise users
  • Show clear error states with retry options
  • Extend session to 7 days with refresh tokens

Technical Approach

We'll use the existing auth service but add a new OAuth flow handler. The token refresh mechanism needs to be updated to support longer sessions without compromising security.
