Issues / PS-201

Simplify password reset flow

Jan 15, 2026
Marcus Chen
To Do
High Priority
You + AI

Problem

Users are abandoning the password reset flow due to complexity. Current flow requires 5 steps and users are dropping off at the password confirmation screen.

Impact

  • ~30% abandonment rate at password reset
  • Increased support tickets about account access
  • User frustration leading to churn

Proposed Solution

Reduce the flow to 2-3 steps with clearer guidance:

  1. Enter email → receive magic link or code
  2. Click link → auto-login with prompt to set new password
  3. Optional: Set new password (or keep using magic links)

Acceptance Criteria

  • Password reset completes in 3 steps or fewer
  • Clear progress indicator throughout flow
  • Error messages are actionable (not just "Something went wrong")
  • Magic link expires after 15 minutes
  • User can resend link without starting over
