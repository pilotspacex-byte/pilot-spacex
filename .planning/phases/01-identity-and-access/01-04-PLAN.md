    - `SessionRecordingMiddleware(BaseHTTPMiddleware)` — FastAPI/Starlette middleware
    - On each request:
      1. Extract JWT from Authorization header (Bearer) — skip if not present (unauthenticated routes)
      2. Compute SHA-256 hash of token: `hashlib.sha256(token.encode()).hexdigest()`
      3. Extract workspace_id from path or request state if available (routes like `/workspaces/{slug}/...`)
      4. Check Redis REVOKED_KEY — if present, return JSONResponse({"detail": "Session has been revoked"}, status_code=401)
      5. Call record_session (throttled via LASTSEEN_KEY) — fire-and-forget with asyncio.create_task to not block request
      6. Continue to next middleware/handler
    - `record_session(redis, db, token_hash, user_id, workspace_id, ip, user_agent)`: check LASTSEEN_KEY; if absent, upsert session + set LASTSEEN_KEY with 60s TTL
    - Parse IP from X-Forwarded-For header first, fallback to request.client.host
    - NOTE: middleware must handle exceptions gracefully — session recording failure must NOT break the request
