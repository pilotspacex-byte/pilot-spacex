**Add quota helper** — Create a `_check_storage_quota(session, workspace_id, content_delta_bytes) -> tuple[bool, float | None]` utility:
- Load workspace by workspace_id
- If `storage_quota_mb` is NULL: return `(True, None)` (unlimited)
- quota_bytes = storage_quota_mb * 1024 * 1024
- projected = storage_used_bytes + content_delta_bytes
- If projected > quota_bytes: return `(False, None)` → caller raises 507
- pct = projected / quota_bytes
- If pct >= 0.80: return `(True, pct)` → caller adds warning header
- Return `(True, None)` → no warning

**Storage delta update** — Add `_update_storage_usage(session, workspace_id, delta_bytes: int)` utility:
```python
await session.execute(
    update(Workspace)
    .where(Workspace.id == workspace_id)
    .values(storage_used_bytes=Workspace.storage_used_bytes + delta_bytes)
)
```

**Quota enforcement in write paths** — Add quota check to note creation and issue creation endpoints in workspaces.py (or the relevant note/issue routers if they live in separate files). Before writing:
```python
content_len = len(body or "")
```
