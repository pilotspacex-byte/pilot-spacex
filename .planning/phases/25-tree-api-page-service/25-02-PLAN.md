1. **reorder_page_service.py**: Create `ReorderPagePayload` (frozen dataclass: note_id UUID, insert_after_id UUID|None, workspace_id UUID, actor_id UUID), `ReorderPageResult` (frozen dataclass: note Note), and `ReorderPageService` class.

   Constructor: `__init__(self, session: AsyncSession, note_repository: NoteRepository)` storing as `_session` and `_note_repo`.

   `async def execute(self, payload: ReorderPagePayload) -> ReorderPageResult`:
   - Fetch note by ID, validate exists and workspace matches
   - Guard: if `note.project_id is None`, raise `ValueError("Personal page reordering not yet supported")`
   - Get siblings via `_note_repo.get_siblings(note.parent_id, payload.workspace_id, note.project_id, note.id)`
   - Compute new position via `_compute_insert_position(siblings, payload.insert_after_id)`
   - If position == -1 (gap exhausted sentinel): call `_resequence_siblings(siblings, note, payload.insert_after_id)`
   - Else: set `note.position = new_position`
   - `await self._session.flush()` + `await self._session.refresh(note)`
   - Return `ReorderPageResult(note=note)`

   `def _compute_insert_position(self, siblings: Sequence[Note], insert_after_id: UUID | None) -> int`:
   - Follow the exact algorithm from RESEARCH.md Pattern 4 (midpoint computation with -1 sentinel for gap exhaustion)

   `async def _resequence_siblings(self, siblings: Sequence[Note], note: Note, insert_after_id: UUID | None) -> None`:
   - Build ordered list: all siblings + note, ordered by current position
   - Insert note at correct position (after insert_after_id, or at head if None)
   - Assign positions: 1000, 2000, 3000, ... to each in order
   - Bulk update all via individual `sibling.position = new_pos` + single flush

2. Add helper `_note_to_tree_response(note: Note) -> PageTreeResponse` that mirrors `_note_to_response` but includes `parent_id`, `depth`, `position`.

3. Add `POST /{workspace_id}/notes/{note_id}/move` endpoint:
   ```python
   @router.post(
       "/{workspace_id}/notes/{note_id}/move",
       response_model=PageTreeResponse,
       tags=["workspace-notes"],
   )
   async def move_page(
       workspace_id: WorkspaceIdOrSlug,
       note_id: NoteIdPath,
       body: MovePageRequest,
       current_user_id: CurrentUserId,
       session: SessionDep,  # CRITICAL: must be in signature for DI ContextVar
       move_service: MovePageServiceDep,
       workspace_repo: WorkspaceRepositoryDep,
   ) -> PageTreeResponse:
       workspace = await _resolve_workspace(workspace_id, workspace_repo)
       try:
           result = await move_service.execute(MovePagePayload(
               note_id=note_id,
               new_parent_id=body.new_parent_id,
               workspace_id=workspace.id,
               actor_id=current_user_id,
           ))
       except ValueError as e:
           raise HTTPException(status_code=status.HTTP_422_UNPROCESSABLE_ENTITY, detail=str(e)) from e
       return _note_to_tree_response(result.note)
   ```

4. Add `POST /{workspace_id}/notes/{note_id}/reorder` endpoint:
   ```python
   @router.post(
       "/{workspace_id}/notes/{note_id}/reorder",
       response_model=PageTreeResponse,
       tags=["workspace-notes"],
   )
   async def reorder_page(
       workspace_id: WorkspaceIdOrSlug,
       note_id: NoteIdPath,
       body: ReorderPageRequest,
       current_user_id: CurrentUserId,
       session: SessionDep,  # CRITICAL
       reorder_service: ReorderPageServiceDep,
       workspace_repo: WorkspaceRepositoryDep,
   ) -> PageTreeResponse:
       workspace = await _resolve_workspace(workspace_id, workspace_repo)

           result = await reorder_service.execute(ReorderPagePayload(
               note_id=note_id,

               actor_id=current_user_id,
           ))
       except ValueError as e:
           raise HTTPException(status_code=status.HTTP_422_UNPROCESSABLE_ENTITY, detail=str(e)) from e
       return _note_to_tree_response(result.note)
   ```
