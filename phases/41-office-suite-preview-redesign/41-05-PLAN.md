    **Schemas** (`artifact_annotations.py` in schemas/):
    ```python
    class CreateAnnotationRequest(BaseModel):
        slide_index: int = Field(ge=0)
        content: str = Field(min_length=1, max_length=5000)

    class UpdateAnnotationRequest(BaseModel):
        content: str = Field(min_length=1, max_length=5000)

    class AnnotationResponse(BaseModel):
        id: str  # UUID as string
        artifact_id: str
        slide_index: int
        content: str
        user_id: str
        created_at: str  # ISO datetime
        updated_at: str

        model_config = ConfigDict(from_attributes=True)

    class AnnotationListResponse(BaseModel):
        annotations: list[AnnotationResponse]
        total: int
    ```

    **Service** (`artifact_annotation_service.py`):
    - Constructor takes `session: AsyncSession` (from DI)
    - `async def create(self, artifact_id: UUID, slide_index: int, content: str, user_id: UUID, workspace_id: UUID) -> ArtifactAnnotation`: Creates annotation via repository
    - `async def list_for_slide(self, artifact_id: UUID, slide_index: int) -> Sequence[ArtifactAnnotation]`: Delegates to repository
    - `async def update(self, annotation_id: UUID, content: str, user_id: UUID) -> ArtifactAnnotation`: Checks ownership (annotation.user_id == user_id), raises HTTPException(403) if not owner. Updates content.
    - `async def delete(self, annotation_id: UUID, user_id: UUID) -> None`: Checks ownership, calls `repository.hard_delete()`. Raises 403 if not owner, 404 if not found.

    **Router** (`artifact_annotations.py` in routers/):
    Use `APIRouter()`. All endpoints require `session: SessionDep` and `user: CurrentUser` and `require_workspace_member`.

    Base path will be mounted at: `/workspaces/{workspace_id}/projects/{project_id}/artifacts/{artifact_id}/annotations`

    Endpoints (use `""` not `"/"` for root per CLAUDE.md gotcha):
    - `POST ""` -- Create annotation. Body: CreateAnnotationRequest. Returns AnnotationResponse (201).
    - `GET ""` -- List annotations. Query param: `slide_index: int | None = None`. Returns AnnotationListResponse.
    - `PUT "/{annotation_id}"` -- Update annotation. Body: UpdateAnnotationRequest. Returns AnnotationResponse. 403 if not owner.
    - `DELETE "/{annotation_id}"` -- Delete annotation. Returns 204. 403 if not owner. 404 if not found.

    Each endpoint: instantiate `ArtifactAnnotationRepository(session, ArtifactAnnotation)` and `ArtifactAnnotationService(repository)` inline (or via DI). Call `set_rls_context(session, workspace_id, user.id)` before any DB operation.
