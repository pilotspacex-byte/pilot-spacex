**State:**
- `status`: `'loading' | 'rendered' | 'error'`
- `headings`: `TocHeading[]`
- `iframeRef`: `React.RefObject<HTMLIFrameElement>`

**Rendering logic** (useEffect, depends on `data`):
1. Guard: `if (!iframeRef.current) return;`
2. Set `status = 'loading'`
3. Get iframe's `contentDocument.body` — clear it using `doc.body.textContent = ''`
4. Try `docx-preview.renderAsync`:
   ```typescript
   const { renderAsync } = await import('docx-preview');
   await renderAsync(data, iframeDoc.body, undefined, {
     className: 'docx-preview',
     inWrapper: true,
     ignoreWidth: false,
     ignoreHeight: false,
     breakPages: true,
   });
   ```
5. On catch, try mammoth fallback with DOMPurify sanitization:
   ```typescript
   const mammoth = (await import('mammoth')).default;
   const DOMPurify = (await import('dompurify')).default;
   const result = await mammoth.convertToHtml({ arrayBuffer: data });
   const sanitized = DOMPurify.sanitize(result.value);
   iframeDoc.body.textContent = ''; // Clear safely
   const template = iframeDoc.createElement('template');
   template.innerHTML = sanitized; // Safe: DOMPurify-sanitized content
   iframeDoc.body.appendChild(template.content);
   ```
6. On both catch: set `status = 'error'`
7. On success: set `status = 'rendered'`
8. Extract headings from iframe:
   ```typescript
   const extractedHeadings = Array.from(iframeDoc.querySelectorAll('h1, h2, h3, h4, h5, h6')).map((el, i) => {
     const id = el.id || `heading-${i}`;
     if (!el.id) el.id = id;
     return { id, text: el.textContent?.trim() ?? '', level: parseInt(el.tagName[1]) };
   });
   setHeadings(extractedHeadings);
   ```

**onScrollToHeading callback** (passed to DocxTocSidebar):
```typescript
const handleScrollToHeading = React.useCallback((headingId: string) => {
  const iframeDoc = iframeRef.current?.contentDocument;
  if (!iframeDoc) return;
  const el = iframeDoc.getElementById(headingId);
  if (!el) return;
  el.scrollIntoView({ behavior: 'smooth', block: 'start' });
  el.classList.add('highlight-heading');
  setTimeout(() => el.classList.remove('highlight-heading'), 2000);
}, []);
```

```typescript
export function DocxTocSidebar({ headings, onScrollToHeading }: DocxTocSidebarProps) {
  const [activeId, setActiveId] = React.useState<string | null>(null);

  if (headings.length === 0) {
    return (
      <div className="w-56 shrink-0 border-r border-border/60 p-4">
        <p className="text-xs text-muted-foreground">No headings found</p>
      </div>
    );
  }

  return (
    <div className="w-56 shrink-0 border-r border-border/60 flex flex-col">
      <div className="px-3 py-2 border-b border-border/60">
        <p className="text-[10px] font-semibold uppercase tracking-wider text-muted-foreground/70">
          Contents
        </p>
      </div>
      <ScrollArea className="flex-1">
        <nav className="p-2" aria-label="Table of contents">
          {headings.map((h) => (
            <button
              key={h.id}
              onClick={() => {
                setActiveId(h.id);
                onScrollToHeading(h.id);
              }}
              className={cn(
                'block w-full text-left rounded px-2 py-1 transition-colors hover:bg-muted/50',
                activeId === h.id && 'bg-muted/60',
                h.level === 1 && 'text-[13px] font-semibold text-foreground/80',
                h.level === 2 && 'text-[13px] pl-4 text-muted-foreground',
                h.level >= 3 && 'text-xs pl-7 text-muted-foreground/70',
              )}
            >
              <span className="line-clamp-2">{h.text}</span>
            </button>
          ))}
        </nav>
      </ScrollArea>
    </div>
  );
}
```

Key details per UI-SPEC:
- Width: `w-56` (224px)
- Section header: `text-[10px] font-semibold uppercase tracking-wider text-muted-foreground/70` with text "Contents"
- h1: `text-[13px] font-semibold text-foreground/80`
- h2: `text-[13px] pl-4 text-muted-foreground`
- h3+: `text-xs pl-7 text-muted-foreground/70`
- Empty state: "No headings found"
- Active heading gets `bg-muted/60` highlight
- Each heading button is `line-clamp-2` to truncate long headings
