## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, resize handle width, sheet tab gaps |
| sm | 8px | Compact element spacing, toolbar gaps |
| md | 16px | Default element spacing, iframe body padding, canvas padding |
| lg | 24px | Section padding, mammoth iframe horizontal padding |
| xl | 32px | Layout gaps, page break margins |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Not used in this phase |

Exceptions:
- Touch targets for toolbar buttons: 28px (`size-7`) for compact toolbar, 32px (`size-8`) for standard toolbar actions
- Thumbnail strip width: 156px (120px canvas + 36px for number + padding)
- Annotation panel width: 320px (`w-80`)
- ToC sidebar width: 224px (`w-56`)

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA (Annotation) | "Add Note" (button) / "Add a note..." (placeholder) |
| Empty state heading (Annotations) | No heading — icon + body only |
| Empty state body (Annotations) | "No annotations on this slide yet." |
| Empty state body (ToC) | "No headings found" |
| Error state (Render failure) | Component renders `DownloadFallback` with contextual reason |
| Error state (PPTX load) | "Failed to load presentation: {error}" |
| Error state (Thumbnails) | "Failed to load thumbnails" |
| Error state (Annotations load) | "Failed to load annotations. Please try again." |
| Destructive confirmation (Delete annotation) | No confirmation dialog — immediate delete with optimistic UI (annotation fades to `opacity-50` during delete) |
| Truncation warning (XLSX) | "Showing 500 of {N} rows. Download for full data." |
| Expired URL | Renders `DownloadFallback` with reason "expired" |
| Legacy format | Renders `DownloadFallback` with reason "legacy" |
| Search feedback | "{N} match" / "{N} matches" (singular/plural) |
| Annotation submit hint | "Cmd+Enter to submit" (shown only when textarea has content) |
| Annotation pending | "Adding..." (button text during mutation) |
| Slide counter format | "{current} / {total}" or "{current} / ..." during load |
| Page break label | "Page break" (CSS `::before` pseudo-element, uppercase, centered on dashed line) |
| DOCX iframe sandbox warning | No visible copy — security is transparent to user |

## Focal Points

Each renderer declares a single visual focal point — the element the user's eye should land on first.

| Renderer | Focal Point | Implementation |
|----------|-------------|----------------|
| PPTX | Canvas (current slide) | Centered in flex container, `shadow-md ring-1 ring-border/40 bg-white rounded-lg` draws the eye; sidebars are muted (`bg-muted/10`, `bg-muted/20`) to recede |
| XLSX | Table body (data cells) | Sticky header provides stable anchor; body fills remaining scroll area with no competing chrome; search toolbar is visually quiet (`bg-muted/20`) |
| DOCX | Iframe (rendered document) | Full `flex-1` width with `bg-white` against `bg-neutral-50` wrapper creates page-on-surface contrast; ToC sidebar is narrow (224px) and muted |

### Tooltip Strategy for Icon-Only Toolbar Buttons

All icon-only buttons in toolbars and panel headers use the shadcn `Tooltip` component (Radix Tooltip primitive):

| Property | Value |
|----------|-------|
| Component | `<Tooltip>` / `<TooltipTrigger>` / `<TooltipContent>` from `@/components/ui/tooltip` |
| Delay | `delayDuration={300}` (300ms open delay) |
| Side | `side="bottom"` for top toolbars, `side="top"` for bottom navigation bars |
| Content style | `text-xs` (12px), default shadcn tooltip styling |
| Keyboard shortcut | Append shortcut in muted parentheses when applicable, e.g. "Previous slide (Left)" |

## Interaction Contracts

### XLSX Interactions

| Interaction | Behavior |
|-------------|----------|
| Column resize (mouse) | Drag resize handle on column header right edge; min width 60px; cursor `col-resize` |
| Column resize (keyboard) | Focus resize handle, Arrow keys adjust width by 5px (20px with Shift) |
| Sheet tab switch | Click tab at bottom; resets search and column widths |
| Search | 300ms debounced input; highlights matches with `<mark>` in cells and headers |
| Frozen header | Sticky `top-0 z-10` with `backdrop-blur-sm` and subtle bottom shadow |
| Alternating rows | Odd rows get `bg-muted/30` |
| Row truncation | Max 500 rows displayed; amber warning banner when truncated |

### DOCX Interactions

| Interaction | Behavior |
|-------------|----------|
| ToC toggle | Button in modal header toggles `tocOpen`; sidebar appears on left (224px wide) |
| ToC heading click | Smooth-scrolls iframe to heading via `contentDocument.getElementById().scrollIntoView()` |
| Heading highlight | 2px solid blue outline for 2 seconds after scroll-to |
| Render fallback | docx-preview failure silently falls through to mammoth; both fail shows DownloadFallback |

### PPTX Interactions

| Interaction | Behavior |
|-------------|----------|
| Slide navigation (arrows) | Left/Right arrow keys when not in textarea/input; wraps at bounds |
| Slide navigation (buttons) | Prev/Next buttons in bottom toolbar; disabled at bounds |
| Slide navigation (thumbnails) | Click thumbnail in left sidebar; ArrowUp/ArrowDown keyboard nav in strip |
| Thumbnail strip toggle | Button in modal header; hidden by default; hidden in fullscreen |
| Fullscreen slideshow | Play button enters Fullscreen API; bottom toolbar becomes floating pill overlay |
| Fullscreen exit | X button in floating toolbar or browser Escape key |
| Canvas resize | ResizeObserver re-renders slide when container width changes; maintains 16:9 aspect |
| Auto-scroll thumbnails | Active thumbnail scrolls into view with `behavior: 'smooth', block: 'nearest'` |
| Lazy thumbnail render | IntersectionObserver with 200px rootMargin; cached after first render |

### Annotation Interactions

| Interaction | Behavior |
|-------------|----------|
| Panel toggle | Click collapsed strip icon to expand; ChevronRight to collapse |
| Create annotation | Type in textarea, click "Add Note" or Cmd+Enter; optimistic temp ID shown at `opacity-50` |
| Edit annotation | Owner-only pencil icon; inline textarea replacement; Save/Cancel actions |
| Delete annotation | Owner-only trash icon; immediate with optimistic `opacity-50` fade; no confirmation dialog |
| Slide change reset | New annotation textarea and edit state reset when slide changes |
| Collapsed badge | Annotation count badge on collapsed icon; `9+` cap |

### Modal Interactions

| Interaction | Behavior |
|-------------|----------|
| Maximize/restore | Toggle between `sm:max-w-3xl max-h-[85vh]` and `w-[96vw] h-[94vh] max-w-none` |
| Close | X button or click outside (with 300ms open debounce to prevent accidental close) |
| Escape | Closes modal (Radix Dialog default) |
| Open animation | `transition-[width,height,max-width] duration-200 ease-out` |

## Layout Contracts

### XLSX Layout

```
+--------------------------------------------------+
| [Search input] [match count]          (toolbar)   |
+--------------------------------------------------+
| Col A  | Col B  | Col C  | Col D  |   (sticky)   |
|--------|--------|--------|--------|              |
| data   | data   | data   | data   |   (scroll)   |
| data   | data   | data   | data   |              |
+--------------------------------------------------+
| Sheet1 | Sheet2 | Sheet3 |         (sheet tabs)   |
+--------------------------------------------------+
```

- Toolbar: `flex items-center gap-2 px-3 py-2 border-b`
- Table: `table-fixed` inside `ScrollArea`
- Header: `sticky top-0 z-10 bg-muted/80 backdrop-blur-sm`
- Sheet tabs: `flex items-center gap-1 px-2 py-1.5 border-t overflow-x-auto`
- Truncation banner: above toolbar when >500 rows

### DOCX Layout

```
+----------+--------------------------------------+
| Contents |                                      |
| h1       |    [iframe: rendered document]       |
|   h2     |                                      |
|     h3   |                                      |
|          |                                      |
+----------+--------------------------------------+
```

- ToC sidebar: `w-56 shrink-0 border-r` (only when `tocOpen`)
- Iframe: `w-full flex-1 border-0 min-h-[500px]`; `sandbox="allow-same-origin"` (no scripts)
- Wrapper: `bg-neutral-50 dark:bg-neutral-950/30` for page-feel depth

### PPTX Layout

```
+--------+---------------------------+-----------+
| 1 [th] |                           | Annot.    |
| 2 [th] |    [canvas: slide]        | panel     |
| 3 [th] |                           | (320px)   |
| 4 [th] |                           |           |
|        +---------------------------+           |
|        | [<] 3 / 12 [>]           |           |
+--------+---------------------------+-----------+
```

- Thumbnail strip: `w-[156px] shrink-0 border-r` (when visible, not in fullscreen)
- Slide canvas: `flex-1 min-w-0 bg-muted/30`; 16:9 aspect ratio maintained
- Canvas container: `rounded-lg overflow-hidden shadow-md ring-1 ring-border/40 bg-white`
- Navigation bar: `flex items-center justify-center gap-1 py-2 px-3 border-t backdrop-blur-sm`
- Annotation panel: `w-80 shrink-0 border-l` (always visible in normal mode, hidden in fullscreen)
- Fullscreen: black background, centered slide max-w-5xl, floating pill navigation at bottom

### PPTX Fullscreen Layout

```
+--------------------------------------------------+
|                                                    |
|                                                    |
|              [canvas: slide]                       |
|              (max-w-5xl, centered)                 |
|                                                    |
|                                                    |
|          [<] 3 / 12 [>] [X]                       |
|          (floating rounded-full pill)              |
+--------------------------------------------------+
```

- Backdrop: `bg-black h-screen w-screen`
- Nav pill: `absolute bottom-4 left-1/2 -translate-x-1/2 bg-black/70 rounded-full border-white/10`
- Controls: white text on semi-transparent background

## Accessibility Contracts

| Requirement | Implementation |
|-------------|---------------|
| Slide thumbnails `role="listbox"` | `aria-orientation="vertical"`, `aria-selected` on active, `role="option"` per item |
| Keyboard slide nav | ArrowLeft/ArrowRight for slides (skipped when in textarea/input) |
| Keyboard thumbnail nav | ArrowUp/ArrowDown within listbox |
| Column resize keyboard | ArrowLeft/ArrowRight on focused separator, Shift for larger steps |
| Loading states | `role="status"` with `aria-label` on all spinners |
| Iframe labeling | `title` and `aria-label` on all iframe elements |
| Resize handle | `role="separator"`, `aria-orientation="vertical"`, `aria-label`, focusable |
| Search input | `aria-label="Search in sheet"` |
| Annotation textarea | `aria-label` on all textareas |
| Modal | Radix Dialog handles `aria-modal`, focus trap, Escape close |
| Screen reader description | `DialogDescription` with `sr-only` class |
| Tooltip accessibility | Radix Tooltip provides `aria-describedby` on trigger automatically |
