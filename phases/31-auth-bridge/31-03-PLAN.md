<objective>
Enable OAuth sign-in (Google, GitHub) that redirects back into the desktop app via the `pilotspace://auth/callback` deep link.

Purpose: In static export mode, Next.js API route handlers (like `app/api/auth/callback/route.ts`) do not exist. The web-based Supabase OAuth callback cannot work. Instead, OAuth opens an external browser, the user authenticates, and Supabase redirects to `pilotspace://auth/callback?code=...`. The Tauri app intercepts this deep link via `tauri-plugin-deep-link`, extracts the authorization code, and calls `supabase.auth.exchangeCodeForSession()` to complete the PKCE flow.

Output: `tauri-plugin-deep-link` configured with `pilotspace` scheme, deep link listener in `tauri-auth.ts`, updated `AuthStore.loginWithOAuth` for Tauri mode, `@tauri-apps/plugin-deep-link` installed in frontend.
</objective>

1. Add `initDeepLinkListener()` function to `frontend/src/lib/tauri-auth.ts`:

This function subscribes to deep link events via `@tauri-apps/plugin-deep-link`. When a `pilotspace://auth/callback?code=...` URL is received, it:
  a. Parses the URL to extract the `code` query parameter
  b. Calls `supabase.auth.exchangeCodeForSession(code)` to complete the PKCE flow
  c. The existing `onAuthStateChange` listener (from `syncTokenToTauriStore`) will automatically sync the new session to Tauri Store and keychain

```typescript
let deepLinkInitialized = false;

export async function initDeepLinkListener(): Promise<void> {
  if (deepLinkInitialized) return;
  deepLinkInitialized = true;

  const { onOpenUrl } = await import('@tauri-apps/plugin-deep-link');

  await onOpenUrl(async (urls: string[]) => {
    for (const rawUrl of urls) {
      try {
        const url = new URL(rawUrl);

        // Only handle auth callback deep links
        if (url.host !== 'auth' || url.pathname !== '/callback') continue;

        const code = url.searchParams.get('code');
        if (!code) {
          console.error('[tauri-auth] Deep link missing code parameter:', rawUrl);
          continue;
        }

        // Exchange the authorization code for a session (PKCE flow completion)
        const { data, error } = await supabase.auth.exchangeCodeForSession(code);

        if (error) {
          console.error('[tauri-auth] Failed to exchange code for session:', error.message);
          // Navigate to login with error
          if (typeof window !== 'undefined') {
            window.location.href = `/login?error=${encodeURIComponent(error.message)}`;
          }
          continue;
        }

        if (data.session) {
          // Session is now active. The onAuthStateChange listener from
          // syncTokenToTauriStore() will automatically sync tokens to Store + keychain.
          // Navigate to the app home.
          if (typeof window !== 'undefined') {
            window.location.href = '/';
          }
        }
      } catch (err) {
        console.error('[tauri-auth] Deep link processing error:', err);
      }
    }
  });
}
```

Update the `loginWithOAuth` method:
```typescript
async loginWithOAuth(provider: 'github' | 'google'): Promise<void> {
  if (isAuthCoreMode) {
    throw new Error('OAuth login is not available with AuthCore');
  }

  this.isLoading = true;
  this.error = null;

  try {
    // In Tauri mode, redirect to deep link scheme so the app intercepts the callback.
    // In web mode, redirect to the existing /auth/callback page.
    const redirectTo = isTauri()
      ? 'pilotspace://auth/callback'
      : `${window.location.origin}/auth/callback`;

    const { error } = await supabase.auth.signInWithOAuth({
      provider,
      options: {
        redirectTo,
        // PKCE flow: skipBrowserRedirect is false (default) — opens external browser
      },
    });

    if (error) {
      runInAction(() => {
        this.error = error.message;
        this.isLoading = false;
      });
    }
    // In Tauri mode, the external browser opens and the app waits for
    // the deep link callback. isLoading stays true until the deep link
    // handler calls exchangeCodeForSession and triggers onAuthStateChange.
  } catch (err) {
    runInAction(() => {
      this.error = err instanceof Error ? err.message : 'OAuth login failed';
      this.isLoading = false;
    });
  }
}
```
