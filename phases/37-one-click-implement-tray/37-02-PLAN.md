2. Create `tauri-app/src-tauri/src/commands/tray.rs`:
   - `use tauri::{AppHandle, Manager, tray::{TrayIconBuilder, MouseButton, MouseButtonState, TrayIconEvent}, menu::{MenuBuilder, MenuItem}};`
   - `use tauri_plugin_notification::NotificationExt;`

   - `pub fn setup_tray(app: &AppHandle) -> Result<(), Box<dyn std::error::Error>>`:
     a. Build a context menu with 2 items:
        - "Show Window" (`show_item`) — calls `app.get_webview_window("main").unwrap().show()` and `.set_focus()`
        - "Quit" (`quit_item`) — calls `app.exit(0)`
     b. Build a `TrayIconBuilder::with_id("main-tray")`
        - `.icon(app.default_window_icon().cloned().expect("no app icon"))` — reuse the app icon
        - `.tooltip("Pilot Space")`
        - `.menu(&menu)`
        - `.menu_on_left_click(false)` — single click on tray icon shows window (not menu)
        - `.on_tray_icon_event(move |tray, event|` — on left click (MouseButton::Left, MouseButtonState::Up): get window via `tray.app_handle().get_webview_window("main")`, call `.show()` and `.set_focus()`
        - `.on_menu_event(move |app, event|` — match event.id: "show" -> show+focus window, "quit" -> app.exit(0))
        - `.build(app)?;`
     c. Return Ok(())

   - `pub fn setup_close_to_tray(app: &AppHandle)`:
     a. Get the main window: `app.get_webview_window("main")`
     b. Listen on the "close-requested" window event: `window.on_window_event(move |event|`
     c. If `event` is `tauri::WindowEvent::CloseRequested { api, .. }`:
        - Call `api.prevent_close()` — prevents actual window close
        - Call `window.hide().unwrap()` — hide window to tray
     d. This makes the close button minimize to tray instead of quitting

   - `#[tauri::command]`
     `pub async fn send_notification(app: tauri::AppHandle, title: String, body: String) -> Result<(), String>`:
     a. Use `app.notification()` (from `NotificationExt`)
     b. `.builder()`
     c. `.title(&title)`
     d. `.body(&body)`
     e. `.show()`
     f. `.map_err(|e| format!("Notification failed: {e}"))?;`
     g. Return Ok(())

1. Add `sendNotification` wrapper to `frontend/src/lib/tauri.ts`:
   ```typescript
   /**
    * Send a native OS notification via the system tray.
    * No-op if not in Tauri mode.
    * @param title - Notification title
    * @param body - Notification body text
    */
   export async function sendNotification(title: string, body: string): Promise<void> {
     if (!isTauri()) return;
     const { invoke } = await import('@tauri-apps/api/core');
     await invoke('send_notification', { title, body });
   }
   ```
   Place it after the existing sidecar commands section.

2. Create `frontend/src/components/desktop/tray-notification-listener.tsx`:
   - 'use client' directive
   - Import useEffect from 'react'
   - Import isTauri from '@/lib/tauri'
   - Component: `TrayNotificationListener` (NOT observer — no MobX state consumed)
   - In useEffect (mount-only, `[]` deps):
     a. If !isTauri(), return early
     b. Define handler for 'implement-complete' CustomEvent:
        ```typescript
        const handler = async (e: Event) => {
          const detail = (e as CustomEvent).detail as {
            issueId: string;
            success: boolean;
            error?: string;
          };
          const { sendNotification } = await import('@/lib/tauri');
          if (detail.success) {
            await sendNotification(
              'Implement Complete',
              `Successfully implemented ${detail.issueId} and pushed to remote.`
            );
          } else {
            await sendNotification(
              'Implement Failed',
              `Failed to implement ${detail.issueId}: ${detail.error || 'Unknown error'}`
            );
          }
        };
        ```
     c. `window.addEventListener('implement-complete', handler)`
     d. Cleanup: `return () => window.removeEventListener('implement-complete', handler)`
   - Render: `return null` (invisible listener component)
