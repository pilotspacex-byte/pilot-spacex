**1. Create `frontend/src/features/git/components/commit-panel.tsx`:**

A self-contained panel component that provides the complete commit workflow.

```typescript
'use client';

import { observer } from 'mobx-react-lite';
import { useState } from 'react';
import { useGitStore } from '@/stores/RootStore';
import { GitCommit, ArrowUp, Loader2, Check, AlertCircle } from 'lucide-react';
import { Button } from '@/components/ui/button';
import { Textarea } from '@/components/ui/textarea';
import { Separator } from '@/components/ui/separator';
```

**CommitPanel structure:**

- Wraps in `observer()` to react to GitStore changes
- Shows a commit message `<Textarea>` with:
  - Placeholder: "Commit message..."
  - Character count below (e.g., "42 characters")
  - Local state via `useState<string>('')` for the message (NOT stored in GitStore — ephemeral UI state)
  - min height 3 rows, max 6 rows
- Below the textarea, a row of action buttons:
  - **Commit** button (primary): disabled when `commitMessage.trim() === ''` or `gitStore.stagedFiles.length === 0` or `gitStore.isCommitting`. Shows `<Loader2 className="animate-spin" />` when committing. onClick: `await gitStore.commit(commitMessage)`, then clear the textarea on success.
  - **Commit & Push** button (outline): same disabled logic as Commit, but after `gitStore.commit()` succeeds, automatically calls `await gitStore.push()`. Shows "Pushing..." state during push phase.
  - **Push** button (ghost, secondary): only visible when `gitStore.status?.ahead > 0` (there are unpushed commits). Disabled when `gitStore.isPushing`. Calls `gitStore.push()`.

- Success state: When `gitStore.lastCommitOid` is set, show a brief success banner with `<Check />` icon and the short OID (first 7 chars): "Committed abc1234". Add a dismiss timeout of 5 seconds using `useEffect` that calls `gitStore.clearCommitState()`.

- Error state: When `gitStore.commitError` is set, show in `text-destructive text-xs` below the buttons.

- Stage summary above the textarea: "N file(s) staged" — clickable count that could scroll to the staged section. Show `gitStore.stageError` in destructive text if set.
