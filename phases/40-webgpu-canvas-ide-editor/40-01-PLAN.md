    - closeFile("f1") removes from openFiles; activeFileId falls back to last remaining tab

    - activeFile computed returns the OpenFile for activeFileId or undefined

    - markClean("f1") sets isDirty=false
    - closeAll() clears all tabs
    - closeOthers("f1") keeps only f1 open

    - MAX_TABS=12; openFile when at 12 tabs closes the oldest non-dirty, non-active tab

  <action>
    1. Create `frontend/src/features/file-browser/stores/FileStore.ts`:
       - Import `makeAutoObservable, computed` from mobx
       - Import `OpenFile` from `@/features/editor/types`
       - Class `FileStore` with:
         - `openFiles: Map<string, OpenFile> = new Map()` (observable)
         - `activeFileId: string | null = null` (observable)
         - `MAX_TABS = 12` (static readonly)
         - Constructor calls `makeAutoObservable(this)`
         - `get activeFile(): OpenFile | undefined` — returns openFiles.get(activeFileId)
         - `get tabs(): OpenFile[]` — Array.from(openFiles.values())
         - `get hasDirtyFiles(): boolean` — tabs.some(f => f.isDirty)
         - `openFile(file: Omit<OpenFile, 'isDirty'>)` — adds to map if not present; if at MAX_TABS, evicts oldest non-dirty non-active; sets activeFileId
         - `closeFile(id: string)` — deletes from map; if was active, set activeFileId to last remaining key or null
         - `closeAll()` — clears map, sets activeFileId to null
         - `closeOthers(keepId: string)` — removes all except keepId; sets activeFileId to keepId
         - `markDirty(id: string)` — sets isDirty=true on file
         - `markClean(id: string)` — sets isDirty=false on file
         - `updateContent(id: string, content: string)` — updates content and marks dirty
