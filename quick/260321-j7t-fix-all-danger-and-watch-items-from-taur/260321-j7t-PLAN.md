`index.add_path()` fails for files that no longer exist on disk. Before calling `add_path`, check if the file exists on disk relative to the repo workdir. If the file does NOT exist, use `index.remove_path()` instead.

Replace the for loop (lines 967-971) with:
```rust
for path in &paths {
    let rel = Path::new(path);
    let abs = Path::new(&repo_path).join(rel);
    if abs.exists() {
        index
            .add_path(rel)
            .map_err(|e| format!("Failed to stage '{}': {}", path, e))?;
    } else {
        index
            .remove_path(rel)
            .map_err(|e| format!("Failed to stage deleted '{}': {}", path, e))?;
    }
}
```
